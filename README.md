#  Dice Rolling Simulator (Python)

##  Overview

This is a simple Python project that simulates rolling 2 dies. Each run generates a random number between 1 and 12.

##  Features

* Random dice roll generation
* Simple and beginner-friendly code
* Easy to run and modify

##  Technologies Used

* Python

##  How to Run

1. Make sure Python is installed
2. Download the file
3. Run this command:

   ```bash
   python dice.py
   ```
